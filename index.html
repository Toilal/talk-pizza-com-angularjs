Avant, on avait JQuery

Ça marchait très bien !

Mais JQuery ...

  • N'aide pas à structurer le code
  • Ne fait que de la manipulation DOM et de l'AJAX
  • Provoque souvent du spaghetti code !

AngularJS, c'est l'anti JQuery

Framework complet

  • Ne dépends pas de JQuery.
  • Permet d'enrichir HTML (directives)
  • Conçu pour consommer une API REST
  • Intègre une gestion de modules
  • Sépare les responsabilités (MVC)
  • Utilise l'injection de dépendance
  • Est facilement testable

Du HTML sous stéroïdes !

    <form id="todo-form" ng-submit="addTodo()">
        <input id="new-todo" placeholder="What needs to be done?" ng-model="newTodo" ng-disabled="saving" autofocus>

    <section id="main" ng-show="todos.length" ng-cloak>

Les controllers

Le controller contient les données du modèle et publie les fonctions utiles pour la vue.

html
<div ng-controller="ResetCtrl as ctrl">
    <input ng-model="value">
    <button ng-click="ctrl.reset()">Reset</button>
    <span class="small">{{ctrl.value}}</span>
</div>
javascript
.controller('ResetCtrl', function() {
    this.value = 'lorem ipsum'; // Mise à jour automatiquement par binding

    this.reset = function() {
        this.value = '';
    }
});

Les directives

Elles encapsulent un modèle, un controlleur, et une vue (template).

javascript (directive)
.directive('resetInput', function() {
    // Corresponds à l'élément <reset-input></reset-input>
    return {
        restrict: 'E', // tag uniquement
        scope: { // => modèle
            label: '@', // attribut
            value: '=', // expression
            resetValue: '=?' // expression (optionel)
        },
        templateUrl: 'examples/reset.html', // => vue
        controller: 'ResetDirectiveCtrl', // => controller
        controllerAs: 'ctrl' // Nom du controleur au sein du template
    };
});
examples/reset.html (template)
<input ng-model="ctrl.value">
<button ng-click="ctrl.reset()">{{label}}</button>
<span class="small">{{ctrl.value}}</span>

Les directives

javascript (controller)
.controller('ResetDirectiveCtrl', function($scope) {
    // $scope est obtenu par injection de dépendance
    if (!$scope.value) {
        $scope.value = 'lorem ipsum';
    }

    if (!$scope.resetValue) {
        $scope.resetValue = '';
    }

    this.value = $scope.value;

    this.reset = function() {
        this.value = $scope.resetValue;
    }
})

Les directives

Il existe de nombreuses directives intégrées à AngularJS

html
<div ng-controller="AngularDirectivesCtrl as ctrl">
    <div class="reset-input-container"
         ng-repeat="component in ctrl.components">
         ng-class="{'component-selected': component == ctrl.selectedComponent, 'hidden': component.hidden}"
         ng-click="ctrl.handleClick(component)"
         <reset-input label="{{component.label}}"
                      value="component.value"
                      ng-hide="component.hidden">
         </reset-input>
         <span class="big" ng-show="component.hidden">Composant {{component.label}} masqué</span>
    </div>
</div>
javascript
.controller('AngularDirectivesCtrl', function() {
    this.components = [
        {label: '#1', value: 'Premier', resetValue: 'Le numero 1'},
        {label: '#2', value: 'Deuxième', resetValue: 'Le numero 2'},
        {label: '#3', value: 'Troisième', resetValue: 'Le numero 3', hidden: true},
        {label: '#4', value: 'Quatrième', resetValue: 'Le numero 4'}
    ];

    this.handleClick = function(component) {
        // On sélectionne le composant s'il n'est pas déja selectionné, sinon le le déselectionne.
        this.selectedComponent = component != this.selectedComponent ? component : null;
    }
});
